# NEXUS 프로젝트 문서

이 디렉토리는 NEXUS 프로젝트의 설계 문서와 관련 자료를 포함합니다.

## 디렉토리 구조

### `/design` - 설계 문서
프로젝트의 전체적인 설계 문서, 요구사항 명세, 시스템 아키텍처 등

### `/architecture` - 아키텍처 문서
시스템 구조, 컴포넌트 설계, 데이터 플로우 등

### `/api` - API 문서
API 명세, 엔드포인트 정의, 요청/응답 스키마 등

### `/python` - Python 코드 및 문서
Python 관련 코드, 스크립트, 라이브러리 문서 등

## 문서 업로드 가이드

1. 설계 문서는 `/design` 디렉토리에 업로드
2. 시스템 아키텍처 문서는 `/architecture` 디렉토리에 업로드
3. API 관련 문서는 `/api` 디렉토리에 업로드
4. Python 코드 및 문서는 `/python` 디렉토리에 업로드
