## ❓ FAQ

### Q1: 플레이스홀더로 시작해도 나중에 교체 가능한가요?
**A**: ✅ 가능합니다. `Live2DPlaceholder` → `Live2DManager`로 클래스만 교체하면 됩니다. 나머지 코드는 동일합니다.

### Q2: GIF vs Live2D 실제 모델, 차이가 얼마나 나나요?
**A**: 
- **GIF**: 정해진 애니메이션만 반복 (3-5초 루프)
- **Live2D**: 실시간 상호작용 (마우스 추적, 립싱크, 파라미터 조절)
- **체감 차이**: 약 70% (GIF도 충분히 좋지만 Live2D가 훨씬 생동감 있음)

### Q3: 외주 제작 시 어떤 플랫폼이 좋나요?
**A**:
- **Fiverr**: 영어 소통, $300-500, 2-3주
- **BOOTH (일본)**: 기존 모델 구매, $150-300, 즉시
- **국내 외주**: 네이버 크몽, 높은 가격 ($500-1000)

### Q4: SDK 통합이 복잡한가요?
**A**: 중간 난이도. `pixi-live2d-display` 라이브러리 사용 시 비교적 간단합니다 (6-8시간).

### Q5: 모바일에서도 잘 작동하나요?
**A**: ✅ 가능합니다. WebGL 지원 필수, 저사양 기기는 플레이스홀더로 폴백.
